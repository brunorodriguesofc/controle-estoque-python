# Sistema Simples de Estoque em Python

Projeto simples desenvolvido em Python para simular um pequeno sistema de estoque de loja utilizando **dicionários**.

O objetivo do projeto é praticar conceitos básicos da linguagem e manipulação de estruturas de dados.

## Funcionalidades

O programa permite:

- Consultar um produto no estoque
- Exibir o preço de um produto
- Atualizar o preço de um item
- Adicionar um novo produto ao estoque
- Remover um produto do dicionário
- Analisar o estoque (quantidade de produtos, lista de produtos e preços)

## Tecnologias utilizadas

- Python
- Estrutura de dados: dicionários

## Conceitos praticados

Durante o desenvolvimento foram utilizados conceitos como:

- `input()` para interação com o usuário
- manipulação de **dicionários**
- métodos como `keys()`, `values()` e `pop()`
- verificação de existência com `in`
- uso de `len()` para contagem de itens
- formatação de valores com `f-strings`
